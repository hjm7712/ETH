# ETH WATCH 실행 방법 (한국어 메뉴얼)

이더리움 실시간 신호 대시보드. 가격·신호·거래량·뉴스를 외부 API에서 실시간으로 받아옵니다.

---

## ✅ 가장 쉬운 방법 — 더블클릭 한 번 (Windows)

1. 폴더 안의 **`start.bat`** 파일을 **더블클릭**
   - **Node.js가 없으면 자동으로 설치**합니다 (winget 사용, 관리자 승인창에서 "예")
     - 자동 설치가 끝난 뒤 "다시 더블클릭하라"는 안내가 나오면, 창을 닫고 `start.bat`을 한 번 더 더블클릭하세요 (PATH 인식용)
   - 처음 한 번은 패키지도 자동 설치합니다 (1~2분, 인터넷 필요)
   - 준비가 끝나면 서버가 켜지고 **브라우저가 자동으로 열립니다**
2. 끝! 자동으로 **http://localhost:3000** 이 열립니다.

> Node.js 자동 설치가 막히면(winget 없음 등) https://nodejs.org 에서 **LTS** 버전을 직접 설치 후 다시 실행하세요.

> 종료하려면 검은 명령창에서 **Ctrl+C** 를 누르거나 창을 닫으세요.
> 브라우저가 "연결 실패"로 뜨면 5~10초 뒤 **새로고침(F5)** 하세요. 서버가 켜지는 중입니다.

## 💻 수동 실행 (Mac/Linux 포함)

터미널에서:
```bash
cd eth-watch
npm install     # 최초 1회 (패키지 설치)
npm run dev     # 실행
```
브라우저에서 http://localhost:3000 접속.

더 가볍게 쓰려면(최적화 빌드):
```bash
npm run build
npm start
```

## 📦 다른 컴퓨터로 옮길 때

1. 이 폴더를 통째로 복사 (단, **`node_modules`** 와 **`.next`** 폴더는 빼고 — 용량 크고 자동 재생성됨)
2. 옮긴 PC에서 `start.bat` 더블클릭 (또는 `npm install` → `npm run dev`)

> GitHub에서 받았다면 `node_modules`는 원래 없으니 그냥 `start.bat` 더블클릭하면 됩니다.

## ⚠️ 꼭 알아둘 점

- **인터넷 연결 필수** — 가격/뉴스/신호를 실시간 API에서 받아옵니다.
- **지역(IP) 주의** — Binance API를 사용하는데, Binance는 **미국 IP를 차단**합니다.
  한국 등 대부분 지역은 정상. 미국에서 돌리면 가격·차트가 안 뜰 수 있어요
  (이 경우 CoinGecko로 교체 필요 — README의 "If the price feed is blocked" 참고).
- 모든 지표·신호는 **투자 조언이 아니라 참고용**입니다.

## ❓ 자주 묻는 질문

**Q. 진짜 .exe 파일은 없나요?**
이 앱은 웹 서버 구조라 단일 .exe로 만들기 어렵습니다. 대신 `start.bat` 더블클릭이
exe와 똑같은 "클릭하면 실행" 역할을 합니다. (바탕화면에 바로가기를 만들어 두면 더 편해요:
`start.bat` 우클릭 → 보내기 → 바탕화면에 바로가기 만들기)

**Q. 포트 3000이 이미 쓰이고 있다고 나와요.**
다른 프로그램이 3000 포트를 쓰는 중입니다. 그 프로그램을 끄거나,
`npm run dev -- -p 3001` 처럼 다른 포트로 실행하세요.

**Q. 창을 닫으면 사이트도 꺼지나요?**
네. 검은 명령창이 곧 서버입니다. 창을 닫으면 사이트도 멈춥니다. 다시 쓰려면 `start.bat` 재실행.
